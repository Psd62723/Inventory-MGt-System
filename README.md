## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/Psd62723/Inventory-MGt-System.git
cd Inventory-MGt-System
```

---

### 2️⃣ Install dependencies

#### Backend:

```bash
cd backend
npm install
```

#### Frontend:

```bash
cd frontend
npm install
```

---

### 3️⃣ Run the project

#### Start backend:

```bash
cd backend
npm start
```

#### Start frontend:

```bash
cd frontend
npm start
```

---

## 🔐 Environment Variables

Create a `.env` file in the backend folder:

```env
PORT=5000
DB_URI=your_database_connection_string
```
